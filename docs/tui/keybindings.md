# TUI Keybindings

## Mode Switching

| Key | Action |
|-----|--------|
| `i` | Enter insert mode (edit query) |
| `Esc` | Return to normal mode |

## Query Execution

| Key | Action |
|-----|--------|
| `Enter` | Execute query |

## Navigation (Normal Mode)

### Results Scrolling

| Key | Action |
|-----|--------|
| `j` | Scroll down one row |
| `k` | Scroll up one row |
| `h` | Scroll left |
| `l` | Scroll right |
| `Ctrl+d` | Page down |
| `Ctrl+u` | Page up |
| `g` | Go to first row |
| `G` | Go to last row |

### Focus

| Key | Action |
|-----|--------|
| `Tab` | Switch focus between editor and results |

## Application

| Key | Action |
|-----|--------|
| `:q` | Quit |
| `:wq` | Save and quit |
| `Ctrl+c` | Cancel / Exit |

## Insert Mode

When in insert mode, standard text editing applies:

| Key | Action |
|-----|--------|
| Arrow keys | Move cursor |
| `Backspace` | Delete character |
| `Enter` | Execute query and return to normal mode |
| `Esc` | Cancel and return to normal mode |
